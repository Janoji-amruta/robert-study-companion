# Robert AI
Your AI Study Companion
Robert AI is a premium study-planning platform designed to help
students stay focused, organized, and motivated.
Built with beautiful animations, glassmorphism UI, and an interactive 
AI companion named Robert, the platform generates personalized study 
schedules, revision plans, and exam preparation strategies.
## Features

✨ Animated AI Companion

📚 Personalized Study Plans

🗓 Exam Preparation Schedules

📈 Progress Tracking

🎯 Revision Planning

⚡ Smooth Premium Animations

🌙 Modern Dark UI

📱 Fully Responsive Design

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Local Storage
- SVG Animations
- Glassmorphism Design
